# ChiliDetect video background

Place the optimized 3D chili loop here:

- `chili-3d.webm` for Chromium-based browsers.
- `chili-3d.mp4` for Safari and broader fallback.

Recommended export:

- Duration: 10-20 seconds, seamless loop.
- Size target: 5-10 MB maximum.
- Resolution: 1920x1080 or 2560x1440 if compression stays light.
- Prompt: "3D animated red chili pepper rotating slowly on dark background, cinematic warm rim lighting, hyperrealistic product render, shallow depth of field, seamless loop, 4K".

The auth page automatically falls back to a CSS-rendered cinematic chili sculpture when the video is missing, fails to load, or the user enables reduced motion.
